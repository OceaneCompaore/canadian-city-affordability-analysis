Project Overview
This project analyzes the affordability of major Canadian cities for young graduates by combining economic indicators such as average employment income, monthly rent, cost of living, and unemployment rate.
The goal is to identify which cities offer the best balance between income and living expenses at the beginning of a professional career.

Research Questions
Which Canadian cities offer the best balance between affordability and employment opportunities?
Are the most expensive cities necessarily the best places to start a career?

Data Cleaning & Transformation

Dataset
The dataset was created by combining multiple sources including:
Income data
Rent data
Cost of living index
Unemployment rates
All datasets were cleaned and standardized using SQL before being merged into a final dataset.

Data Cleaning process
Standardized city names (e.g., Montréal → Montreal)
Converted values to numeric format
Removed unnecessary columns
Filtered to keep only common cities across datasets
Merged datasets into a single table
